# Ballu ASP-100 Integration for Home Assistant

Интеграция для управления кондиционером Ballu ASP-100 через Home Assistant.

## Установка

### Через HACS (рекомендуется)

1. Добавьте этот репозиторий в HACS как пользовательский репозиторий
2. Найдите "Ballu ASP-100" в HACS
3. Установите интеграцию
4. Перезагрузите Home Assistant

### Вручную

Скопируйте папку `ballu_asp100` из `custom_components` в папку `custom_components` вашего Home Assistant.

## Настройка

1. Перейдите в Настройки → Устройства и службы → Интеграции
2. Нажмите "Добавить интеграцию"
3. Найдите "Ballu ASP-100"
4. Введите IP-адрес вашего устройства и порт (по умолчанию 80)

## Поддерживаемые функции

- Регулировка температуры
- Изменение режимов работы (охлаждение, обогрев, вентиляция, осушение)
- Регулировка скорости вентилятора
- Управление вертикальными жалюзи

## Важно

Эта интеграция предполагает, что устройство Ballu ASP-100 имеет сетевой интерфейс и доступно по IP-адресу. Возможно, потребуется дополнительная настройка протокола связи.
